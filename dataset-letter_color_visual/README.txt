-- Dataset #2: Letter-Color 2AFC task --

The experiment administered to the participants can be found in 'dataset-letter_color_visual/experiment'.

Run analysis from 'dataset-letter_color_visual/analysis/run_analysis_visual.py'
Participant information can be found in 'dataset-letter_color_visual/analysis/participants_visual.csv'
Copies of the processed data files for the current article can be found in 'dataset-letter_color_visual/derivatives/data_frames'

To analyze the EyeLink data, you need to have the edf2asc executable file, install desktop application.
See here: https://www.sr-research.com/software/

Notes for reading trials in the EyeLink data file:
Trials are marked by a 1 at the index of the onset of each event with respect to the entire pupil time series.
